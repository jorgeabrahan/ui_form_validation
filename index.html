JavaScript UI Form Validation Library

Extend your html form validations easily with HTML attributes!

Keep in mind that this are just UI validations and they shouldn't be trusted to sent data to the backend without a proper validation

Try it out

Letters

Only letters with spaces will be allowed. You won't be able to type symbols nor numbers

Only letters without spaces will be allowed. You won't be able to type symbols nor numbers

Only letters with spaces will be allowed. You won't be able to type symbols, numbers or more than 20 characters

Only capital letters will be allowed. You won't be able to type symbols, numbers or lowercase letters

Only capital letters with no spaces will be allowed. You won't be able to type symbols, numbers or lowercase letters

Only lowercase letters will be allowed. You won't be able to type symbols, numbers or lowercase letters

Only lowercase letters with no spaces will be allowed. You won't be able to type symbols, numbers or lowercase letters

Symbols

Only symbols will be allowed. You won't be able to type letters nor numbers

Numbers

Only int numbers will be allowed. You won't be able to type letters nor symbols or float numbers

Only int positive numbers will be allowed. You won't be able to type letters, symbols, or int negative numbers

Only int negative numbers will be allowed. You won't be able to type letters, symbols, or int positive numbers

Only float numbers will be allowed. You won't be able to type letters nor symbols or int numbers

Only float positive numbers will be allowed. You won't be able to type letters, symbols, or float negative numbers

Only float negative numbers will be allowed. You won't be able to type letters, symbols, or float positive numbers

Mix

Only letters and symbols will be allowed. You won't be able to type numbers of any kind

Only letters and symbols will be allowed. You won't be able to type numbers of any kind nor spaces

Only letters and integer numbers will be allowed. You won't be able to type symbols

Only letters and integer numbers will be allowed. You won't be able to type symbols nor spaces

Only letters and float numbers will be allowed. You won't be able to type symbols

Only letters and float numbers will be allowed. You won't be able to type symbols nor spaces
